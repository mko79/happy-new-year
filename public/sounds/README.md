# Sounds Directory

Place your sound files here.

## Wrong Answer Sound

- **File name**: `wrong-answer.mp3` (or `.wav`, `.ogg`)
- **Location**: `/public/sounds/wrong-answer.mp3`
- **Usage**: Plays when user selects a wrong match in the vocabulary matching game

## Recommended Sound Sources

1. **Freesound.org** - https://freesound.org/
   - Search for: "wrong answer", "error", "buzzer", "incorrect"
   - Free with attribution

2. **Zapsplat.com** - https://www.zapsplat.com/
   - Free with account registration
   - Search for: "wrong", "error", "fail"

3. **Pixabay.com** - https://pixabay.com/sound-effects/
   - Free, no attribution required
   - Search for: "wrong answer", "error sound"

4. **Mixkit.co** - https://mixkit.co/free-sound-effects/
   - Free sound effects
   - Search for: "error", "wrong"

## Supported Formats

- `.mp3` (recommended)
- `.wav`
- `.ogg`

## File Size Recommendation

Keep sound files under 500KB for better performance.
